# Plot-Template Beispiele

Diese Galerie zeigt stabil erzeugte Beispielbilder und -dokumente fuer alle `plot-template`-Vorlagen.

Die Bilder liegen unter `docs/examples/plot_templates/`.

## Heating

### `heating-year`

**Beispielbefehl:** `python -m ma_analyse plot-template --template heating-year --variants Dimensionierung --rooms "208 office"`

![heating-year](examples/plot_templates/heating-year.png)

### `heating-month`

**Beispielbefehl:** `python -m ma_analyse plot-template --template heating-month --variants Dimensionierung --rooms "208 office" --month Jul`

### `heating-week`
